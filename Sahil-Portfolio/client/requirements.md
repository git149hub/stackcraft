## Packages
framer-motion | Smooth scroll animations and entry effects
clsx | Utility for constructing className strings conditionally (often used with Tailwind)
tailwind-merge | Utility to merge Tailwind classes efficiently

## Notes
The application uses a "Single Page" portfolio layout.
Nav links will use anchor tags with IDs for smooth scrolling within the page.
Images for projects will use placeholders or Unsplash if specific URLs aren't provided in the DB.
The design focuses on a "Dark Mode" technical aesthetic suitable for a backend/data engineer.
